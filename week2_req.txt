Assignment: Geometry Calculator in C
Course: Introduction to C Programming
Topic: Functions, Object Files, and Linking with Clang
Objective

In this assignment you will build a geometry calculator in C that computes the area of different
shapes.
You will place the function implementations in a separate file and compile them into an object file,
then link them with the main program.
This exercise helps you understand how real C programs are built from multiple source files.
Shapes to Support

Your program must calculate the area of:
Circle
Square
Rectangle
Right-angled Triangle
File Structure

Your program should contain two files only:
main.c
geometry.c

This produces the object file:
geometry.o

Step 2: Link the object file with main.c
Step 3: Run the program
